The posts Collection
Instead of storing the "Score" as just a number, we store the metadata needed to display the post.
{
  "_id": "ObjectId",
  "authorId": "ObjectId(user_id_here)",
  "title": "Statement of Detective Alice Daisy Tonner",
  "content": "Full text of the statement...",
  "tags": ["statement", "detective", "trafficstop"],
  "createdAt": "ISODate",
  "score": 124 
}

 The votes Collection (The "Logic" Hub)
To prevent double-voting and track user interaction, you need a separate collection that links a User to a Post.
{
  "_id": "ObjectId",
  "userId": "ObjectId(user_id_here)",
  "postId": "ObjectId(post_id_here)",
  "voteValue": 1, // 1 for upvote, -1 for downvote
}
